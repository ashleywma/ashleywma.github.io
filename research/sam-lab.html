Synergetic Adaptive Machinas (SAM) Lab

Undergraduate research in microrobotics and simulation. Implementation stays in a private codebase (not published here).

Lab

I worked in the Synergetic Adaptive Machinas Lab (University of Michigan Robotics) under Professor Steven Ceron (lab publications).

What I built

One thread is a PyBullet world that couples global planning with local swarm-like dynamics; the other is bring-up on a real 6-axis arm. Details below match my private repo structure (not hosted here).

Simulation: maze, planner, and swarm

I built a 2D grid maze with a depth-first (recursive backtracker) generator, then extruded walls as rigid boxes in PyBullet so the free space matches the graph I plan on. From that layout I build an adjacency matrix (unit cost on open edges) and run Dijkstra's algorithm (1959, with a binary heap in the implementation) from a corner start to a far corner goal. The resulting path is visualized in the sim and used as the backbone for motion.

A Kuka LBR IIWA model stands in for an external magnetic influence: the end-effector tracks the planned route with inverse kinematics and position-controlled joints while the physics clock steps. Along that trajectory I simulate a swarm of many small sphere "microrobots": each feels an inverse-square-style pull toward the moving field target and pairwise repulsion from neighbors, plus lightweight Kuramoto phase coupling so the group does not collapse into a single point. That is a stylized stand-in for attractive / repulsive magnetic interaction, not a full Maxwell solve.

Hardware: MyCobot

On the real arm I used Elephant Robotics' pymycobot over serial: connect at a fixed baud rate, call sync_send_angles for joint-space trajectories (with focus_all_servos / release_all_servos around moves), and poll Cartesian state for debugging. That stack is separate from the PyBullet scene but follows the same idea: scripted poses and coordinated motion for lab demos rather than closed-loop visual servoing.

Direction

The simulation is a sandbox for steering many small agents along a global route while respecting local spacing: a toy model for routing magnetic microrobots over longer paths, with biomedical-style motivation (for example along vessels in the arm). It is simulation-first and not clinical work.
