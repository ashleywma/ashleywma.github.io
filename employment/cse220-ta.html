Teaching assistant · CSE 220

Michigan State University · East Lansing, MI · Jan 2024 – Apr 2024

Spring 2024 teaching assistant for CSE 220 — Programming in C under Professor Manni Liu (Michigan State University). Course coverage spans procedural C, memory and pointers, structured I/O, core data structures, and introductory algorithmic patterns, with parallel emphasis on version control, shell scripting, and test-driven submission workflows.

Lab section enrollment was 33 students. Responsibilities included supervised programming sessions: stack traces and debugger use, reasoning about allocation and ownership of buffers, control-flow and invariant checks, and verifying behavior against specifications rather than pass/fail on the autograder alone. Labs were coordinated with the weekly lecture topics so assignments reinforced the same toolchain expectations—version history, reproducible builds, and incremental tests where the rubric required them.

Course operations covered graded assignments and exams, rubric application consistent with the instructional staff, and coordination with other TAs on deadlines, common failure modes in submissions, and escalation paths for academic integrity or edge-case grading questions.

Skills

Primary stack: C at assignment depth (memory, pointers, structs, file I/O, lists and trees as assigned), plus version control, Unix shell utilities for build and test automation, and structured manual testing where autograding did not cover corner cases. Instructional support included live code review, explaining compiler and linker diagnostics, and aligning feedback with published rubrics and Professor Manni Liu’s course policies.
